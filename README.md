# JWT Authentication System

This is a basic authentication system built using Nest.js, SQLite, and JWT.

## Project Structure

```
src/
├── auth/
│   ├── auth.controller.ts
│   ├── auth.service.ts
│   ├── auth.module.ts
│   ├── jwt.strategy.ts
│   └── local.strategy.ts
├── user/
│   ├── user.controller.ts
│   ├── user.service.ts
│   ├── user.module.ts
│   └── user.entity.ts
├── main.ts
├── app.module.ts
.env
```
